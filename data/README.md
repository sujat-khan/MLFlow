# 📂 Data Directory

This directory holds shared datasets used across multiple modules.

Most datasets in this course are loaded directly from **scikit-learn** or downloaded programmatically, so this folder may remain empty initially.

## Datasets Used

| Dataset | Source | Used In |
|---------|--------|---------|
| Wine Quality | `sklearn.datasets.load_wine()` | Modules 2-9 |
| Iris | `sklearn.datasets.load_iris()` | Module 1 (intro) |

## Notes

- If you download external datasets, place them here for easy access
- Notebooks reference this directory via relative paths: `../data/`
